# Contributing

Thanks for contributing to PowerGuard AI — a few quick guidelines:

- Please run `npm install` and `npm run check` before opening a PR.
- Keep changes small and focused. Add tests for new logic where possible (Vitest is already configured).
- Follow the project's formatting rules: `npm run format`.
- For major changes, open an issue first to discuss the approach.

Maintainers will review and run CI (type checks + build) on each PR.
